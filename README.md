PhoneTracker Advanced is a powerful command-line tool for phone number 
intelligence. Built on Google's libphonenumber engine, it goes far beyond 
simple validation — identifying carriers across 195+ countries using 500+ 
real prefix mappings, enriching results with full geographic metadata 
(coordinates, timezone, currency, language), and scoring each number with 
a heuristic risk engine that flags VoIP, premium-rate, and suspicious 
patterns.

Features include a beautiful boxed terminal UI, four output formats 
(pretty / table / JSON / CSV), batch file processing with export, 
persistent lookup history, an interactive REPL mode, and optional API 
enrichment via NumVerify and AbstractAPI. Zero required API keys — 
everything works offline out of the box.
